# buyer-valuation
**The Renewals Hub**. A read-only portal that centralizes everything a buyer needs to complete their evaluation of the renewal.

**Contract History** — Past contracts with PDF downloads
2. **Renewal Options** — Current terms and paths forward
3. **Queryable Chatbot** — Linked to qualification context — answers questions without involving the seller
4. **ROI Calculator** — Pre-filled with the buyer's data to support internal justification
5. **Key Seller Contacts** — Named contacts for commercial questions

---

## Features

**The Renewals Hub**

1. **Days Until Renewal** — Large countdown — creates urgency without a cold call
2. **Quick-Renew Button** — One-click renewal when the buyer is ready
3. **YoY Comparison** — What changed in the MSA since last year
4. **Engagement Tracking** — Seller sees buyer activity — know when they're active without a check-in call
5. **New Features Since Last Renewal** — Show what the buyer has missed — builds the renewal case automatically
6. **Clear Per-Unit Pricing** — Current price shown as a large, prominent per-unit number — no ambiguity about what the buyer is paying
